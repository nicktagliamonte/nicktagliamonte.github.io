Books

An extremely long term project to read through a list of books.

The list describes itself better than I could in a short paragraph, and any attempt I made to input a "central question" which would define the list didn't really capture the nature or spirit of the project. Largely it's just a very long list of a set of rabbit holes I'd like to go down. The best core idea is probably "if I have been curious about it then I added a book or 10 that covers it."

Try filtering by category to get a sense of general thru-lines. They contain the major areas this project explores, as well as the deviations from the central question which I'm willing to tolerate (the chess books, for example, because I like to play).

The fiction is mostly sci fi, Greg Egan's bibliography is almost all on the list (the only items that aren't are the ones I read before I made the list). It deviates pretty hard into "r/TrueLit's top-40" because of a sort of feedback loop: the though of "ok, if i'm gonna read 150 books i might as well include some of these top 10 books of western lit." "ok, now it's 200 books. I guess I can afford the top 25." and so on and so forth up to the monstrosity below.

It's a living project which I pretty strongly suspect will go unfinished; for every book I finish it seems like I add five more.

Sorting Visualizer

This is a fairly basic sorting visualizer, a pretty classic-if-outdated CS beginner undertaking but the major advantage of the project is that I can put it in an iframe and show a demo here on this site, rather than linking to a separate page or resorting to screenshots.

You can interact with the visualizer by selecting different sorting algorithms and adjusting the speed. The 'step' button allows you to step through the sorting process one step at a time, which is achieved by having a set of yield statements in the code.

There are details about the efficiency and explainations of each algorithm visible when that algorithm is selectable. This is a shorter writeup, becaues the demonstration is the meat of this project. The yield statements and adjustable speed block this from being a speed or efficiency focused project and so there is not benchmarking included for that reason.

A great deal of this project involved textbook implementation of existing sorting algorithms. The major learning curve was figuring out the yield operation. The basic idea is that the visualizer is a generator, and the yield statements are used to pause the execution of the generator function.
